# 小学生英文朗读辅助工具 V1.0.2 更新介绍

发布日期：2026-06-06

## 修复内容

V1.0.2 修复了手机端前三句可以播放，但第 4 句及以后点击播放按钮无反应的问题。

## 问题原因

V1.0.1 的手机兜底播放使用了远程 TTS 音频接口。经过逐句验证，该接口对前三句返回 `audio/mpeg`，但从第 4 句开始的较长句子会返回 `500 application/json`，手机浏览器拿到的不是音频文件，因此无法播放。

## 解决方案

- 对较长句子的手机兜底播放，不再先请求会失败的整句音频。
- 第 4 句及以后直接使用单词音频队列连续播放。
- 每个单词音频都经过验证可返回 `audio/mpeg`。
- 逐词高亮继续跟随单词播放进度更新。
- 保留短句整句播放策略，前三句体验不变。

## 说明

这是一个手机端兼容性补丁，不改变朗读文本、翻译、拆句内容和页面结构。后续如果接入固定 TTS 音频文件，可以把长句播放从逐词兜底升级为更自然的整句音频。
