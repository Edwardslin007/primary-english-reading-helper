# 小学生英文朗读辅助工具 V1.0.1 更新介绍

发布日期：2026-06-06

## 修复内容

V1.0.1 修复了部分手机浏览器、微信内置浏览器或 Android WebView 中点击播放按钮无反应的问题。

## 问题原因

V1.0 的朗读功能依赖浏览器 Web Speech API。部分手机 WebView 不提供 `speechSynthesis` 能力，页面会进入语音不可用状态，播放按钮显示为灰色，点击后无法发声。

## 解决方案

- Web Speech API 可用时，继续使用浏览器内置英文朗读能力。
- Web Speech API 不可用时，自动切换到 HTTPS 远程音频兜底播放。
- 兜底模式下播放按钮保持绿色可点击。
- 单句朗读、单词查读音、拆句短语朗读仍可使用。
- 逐词高亮在兜底模式下使用音频时长进行估算同步。

## 说明

这是一个兼容性补丁，不改变 V1.0 的核心内容和页面结构。后续如果接入固定 TTS 音频文件，可以进一步提升手机端播放速度和音色稳定性。
