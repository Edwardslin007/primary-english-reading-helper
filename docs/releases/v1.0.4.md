# 小学生英文朗读辅助工具 V1.0.4 更新介绍

发布日期：2026-06-07

## 修复内容

V1.0.4 修复了手机端第 4 句开始点击播放后整句立即变红、但只听到最后一个单词发音的问题。

## 问题原因

前几个补丁仍然依赖手机端实时请求第三方 TTS 音频。微信内置浏览器和部分 Android WebView 对这类音频请求、错误回调和连续播放的处理不稳定，导致长句播放失败后进入异常兜底路径，表现为整句高亮异常和只播放最后一个单词。

## 解决方案

- 预生成所有固定朗读音频，并提交到仓库。
- 句子音频放在 `audio/sentences/`。
- 拆句短语音频放在 `audio/phrases/`。
- 单词查读音音频放在 `audio/words/`。
- 网页端优先播放本地 MP3 文件，不再在用户点击时请求第三方 TTS。
- 本地音频出错时，才尝试浏览器内置 Web Speech 兜底。
- 继续保留逐词高亮，按本地音频时长估算进度。

## 说明

这是一次稳定性修复。当前朗读内容固定，静态音频是比实时 TTS URL 更可靠的方案，特别适合 GitHub Pages、手机浏览器和微信内置浏览器访问。
